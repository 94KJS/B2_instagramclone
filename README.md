## 1. 프로젝트명 : WeDesign
+ SNS를 하는 것은 기본적으로 자신을 표현하는 수단이기도 하지만 '나'를 만들어가는 과정이기도 합니다. 개성있는 개인들이 모인 '우리'를 표현하고
디자인해본다는 의미로 위와같이 정하였습니다.
***
## 2. 프로젝트 정보
+ 목표 : 인스타그램과 유사한 형태의 기본 CRUD 기능을 Django로 구현한다.
+ 개발기간 : 2022.09.29 ~ 2022.10.06
+ 멤버구성 : 김준식, 김동익, 김명현, 김해림, 염보미
  * 팀장 김준식 : 총괄, 지원, 발표
  * 역할 분담 
    - 프로필, 상단바 : 김준식
    - 유저 기능 user : 김해림, 김명현
    - 게시물 기능 post : 김동익, 염보미
